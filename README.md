# PbN_Coding_Assessment
Appointment Booking Web App for Hair Salon.

Build an appointment booking web application for a service-based business such as a car repair shop, hair salon, or any similar appointment-driven service in a domain of your choice.

The application should support two primary user journeys:

On the service provider side, staff should be able to configure what the business offers and when it operates. This includes defining services (for example, a haircut or oil change), the people who perform those services, and their schedules. The provider should also be able to see booked appointments.

On the customer side, users should be able to select a service, choose a date, view available time slots, and book an appointment.

To make the system demonstrable, your application should preload sample data, including existing booked appointments.

Bonus (Optional)

If time permits, add a simple provider dashboard that surfaces insights such as:
Most frequently booked services
Most commonly requested days or times
Customer behavior metrics (for example, time to complete a booking or incomplete/abandoned booking attempt
